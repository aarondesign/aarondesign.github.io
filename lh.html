Luteinizing hormone Test Project

Using the phone camera to read the test strip that absorbs the woman's urine, then the application will show the level of luteinizing hormone to info user when is the best fertile window.

User's Journey

A woman who prepares for pregnancy will try kinds of method. One of that is testing the LH level in the urine. In this project, the users will buy the test strip from the pharmacies in the first step. Before test the urine, they should download the application then input the period data to predict the next cycle. When the test day arrives, the app will push notification to info use doing the test.

The user takes a test strip, immersing in the urine and wait for minutes until the test strip reacted. After that, the user will clip the test strip on the phone to analyze the test strip. In a few seconds, it will show the LH level, peak, high and low. The Peak and high level mean it's an excellent fertile day, do the right things now. Repeat the process in a few months, and the users will welcome a new family member.

Design Concept

The main job on this application is to measure the test strip. I want to let user aware when the measurement process starts naturally. Therefore I use two kinds of background to imply user the status. The grey one shows on the settings page, the calendar page and the onboarding; The dark one used on the measurement process.

Design Review

Designing the flow to guide users to place the strip on the correct area is one of the challenge parts. After the first version released, there are some terrible feedbacks about putting the test strip. I hold the product test to find out some usability issues. In the result, the complaint about placing the test strip is rooted out.
