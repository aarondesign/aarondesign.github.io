Clinical trial management system

This trial is the related program of LH App-2.0 . Testee down the app and log in by trial ID. Trial Manager will observe the testees' behavior by the web system.

Key Flow - Observe testees' results

The manager checks the abnormal testees' list. If testees didn't measure by instructions, the manager will call the testees and guide them to measure. After calling, the manager will mark the testee as contacted and take some note about the testee's status.

Abnormal trial result management


Display the testees who didn't measure by instructions.





Click the call button to mark as contacted




Leave some records about this testee.




If every testees follow the instructions and measure correctly.




Check abnormal testees lists of latest 7 days



Manage Testees - Add, Edit, Drop-out

  1. Add - Create new testee and system will generate the trial Email ID.
  2. Edit - Edit testee's info
  3. Drop Out - Testee quit the trial program.
