Analyze the users' behavior

This web is a system that developed for analyzing the users' information and tested data. The stakeholders can use it to analyze the whole users or look up specific users' data.

Login

Dashboard

Here shows the essential info and trend diagram; users can select different time spans to look up more info.

Records

This page shows the records that ordered in descending power, and the options on the left column can filter it.

Members info

This page shows the users list and its basic info, and the options on the left column can filter it.

Multi-Level Filtering

On the records and members page, there are filter options on the left column, and it shows the number of the result. Select multi-options to find out the specific group users.
