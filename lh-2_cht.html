女性黃體素檢測 App 再設計 - 實現版本

舊版的App上線使用經過了兩年，根據這段期間內收集一些使用者數據、使用者訪談以及內外部的利害關係人回饋資訊，於是將整體的UI做個總整理改善使用經驗。


改版主要的目標為：

  1. 首次使用與註冊流程的改善 - 降低使用者的進入門檻
  2. 重新建立 UI system - 確保使用上的ㄧ致性
  3. 調整 Layout - 重新編排資訊的呈現
  4. 增加輔助功能 - 幫助使用者了解產品

伴隨改版完成的事項：

  1. 產品介紹頁面的再設計

首次與註冊使用流程的改善

減少使用者首次使用的門檻


TL;DR

上一個版本的首次使用 ( Onboarding ) 流程相當的繁瑣，包含住註冊、填寫經期資訊、個人基本資料。步驟之多，尤其當填寫完帳號之後需要離開App，打開信箱App查閱驗證碼，接著再回到App。 這段過程需要花費許多時間，有些沒有耐性的使用者應該已經悻悻然的關掉App。

降低門檻

首次使用只詢問兩項必要資訊：週期天數以及上次經期的初始日，這兩個資訊為系統計算受孕期的必要條件。
而註冊流程以及基本資料的填寫流程，將他置於使用者使用系統後，滿滿的引導使用者註冊並填寫基本資料
